MÈTODE de l’objecte canvas.getContext

createLinearGradient()

Crea un degradat de línia. Es pot fer servir per omplir rectangles, cercles, línies, text, etcètera.

Aquest objecte és pot fer servir com a valor de la propietat strokeStyle o la propietat fillStyle, vistes anteriorment.

Es pot fer servir el mètode addColorStop() per indicar diferents colors del degradat i en quina posició situar els colors.

SINTAXÍ

context.createLinearGradient(x0,y0,x1,y1);

Paràmetres

x0 Coordenada x on comença el degradat
y0 Coordenada y on comença el degradat
x1 Coordenada x on acaba el degradat
y1 Coordenada y on acaba el degradat

addColorStop()

Especifica el color on s’ha d’aturar el degradat.
Aquest mètode s’ha d’utilitzar una vegada s’ha fet servir un dels dos mètodes següents: createLinearGradient() o createRadialGradient(). Si no s’utilitza, el degradat no serà visible. S’ha d’utilitzar com a mínim 2 cops.

SINTAXÍ

	gradient.addColorStop(stop,color);

Paràmetres

stop = valor entre 0.0 i 1.0, representa la posició entre el principi i el final del degradat
color = El valor del color CSS que es mostrarà a la posició indicada en l’stop

Exemple: Degradat lineal horitzontal 

		
		let ctx= canvas.getContext("2d");
		let gradient = ctx.createLinearGradient(0,0,0,170);
		gradient.addColorStop(0,'white'); // de blanc a dalt
		gradient.addColorStop(1,'red'); // cap a vermell
		ctx.fillStyle = gradient; // estil de color de dins
		ctx.fillRect(0,0,200,170);


Exemple: Degradat lineal vertical 

		
		let ctx= canvas.getContext("2d");
		let gradient = ctx.createLinearGradient(0,0,170,0);
		gradient.addColorStop(0,'black'); // de negre a l'esquerra
		gradient.addColorStop(0.5,'red'); // cap a vermell
		gradient.addColorStop(1,'white'); // cap a blanc
		ctx.fillStyle = gradient; // estil de color de dins
		ctx.fillRect(0,0,200,150);


Exemple: Degradat lineal horitzontal del marge 

		
		let ctx= canvas.getContext("2d");
		let gradient = ctx.createLinearGradient(0,0,170,0);
		gradient.addColorStop(0,'magenta'); // de magenta a l'esquerra
		gradient.addColorStop(0.5,'blue'); // cap a blau
		gradient.addColorStop(1,'red'); // cap a vermell a la dreta
		ctx.fillStyle = 'pink'; // estil de color de dins
		ctx.fillRect(0,0,200,150);
		// omplir amb el degradat
		ctx.strokeStyle = gradient;
		ctx.lineWidth = 5;
		ctx.strokeRect(3,3,195,144);
